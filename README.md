# ESP32 Smart Clock

## 📌 Project Description
This project is a real-time digital clock built using ESP32 and DS3231 RTC module. It displays current date and time on a 16x2 LCD and includes an alarm system using a buzzer.

## ⚙️ Features
- Real-time clock using DS3231 RTC
- LCD display (16x2 with I2C)
- Alarm system using buzzer
- Accurate time even after power off

## 🧰 Components Used
- ESP32 Microcontroller
- DS3231 RTC Module
- 16x2 LCD Display (I2C)
- Buzzer
- Breadboard & Jumper Wires

## 🔌 Working
ESP32 communicates with RTC module using I2C protocol to get real-time data. The time and date are displayed on LCD. When the set alarm time matches, buzzer turns ON.

## 📊 Applications
- Digital Clock
- Smart Alarm System
- IoT Projects
